Projeto Monitor de Permanência de Clientes

Você è um desenvolvedor especialista em PHP orientado a objetos com MVC.

Preciso que você desenvolva um sistema para monitoramento de permanência de clientes na base de assinantes pelo período de 120 dias. O objetivo é acompanhar os pagamentos mensais da fatura dos próximos quatro meses a partir da data de habilitação do produto.

O sistema deve utilizar PHP orientado objetos com MVC e banco de dados MySQL Que tenha as seguintes características.

- O sistema deve utilizar html com bootstrap para garantir um visual elegante e responsivo
- Deve ter uma tela de login e uma opção de recuperação de senha
- Deve ter três níveis de acesso (operador, coordenador e administrador)
- Deve ter um formulário para cadastro e edição de usuários visível somente para coordenador e administrador
- Deve ter uma tela para cadastro e edição de clientes com nome, CPF com validação, e-mail, contato 1, contato 2.
- 
- Deve ter uma tela para cadastro e edição de proposta de venda que permite adicionar um cliente cadastrado
- Deve ter um Dashboard que mostre um relatório de vendas por período
- O Dashboard também deve mostrar os resultados financeiros
- O Dashboard também deve mostrar uma projeção de lucratividade com o estoque disponível baseado no preço de venda de cada produto menos o preço de compra

Faça a modelagem de dados e crie o banco de dados seguindo as melhores práticas de arquitetura de dados aplicando as formas normais

Após essas etapas estarem funcionando corretamente abordaremos novas funcionalidades.

Pense profundamente sobre todas as etapas necessárias para projetar a arquitetura do sistema seguindo as melhores de programação.
